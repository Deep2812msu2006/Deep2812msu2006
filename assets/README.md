# Assets Directory

This directory contains auto-generated SVG cards for the GitHub profile README.

The following files are automatically generated by the `.github/workflows/generate-cards.yml` workflow:

- `stats.svg` — GitHub stats card
- `top-langs.svg` — Top languages card
- `streak.svg` — Contribution streak card
- `activity-graph.svg` — Contribution activity graph
- `trophies.svg` — GitHub trophies

## How to Regenerate

1. Go to your repo's **Actions** tab
2. Select **Generate README Cards**
3. Click **Run workflow**

The workflow also runs automatically once per day.
